
# 💸 Expense Tracker Android App

An Android application built using Java and Android Studio to manage daily expenses, track income, and monitor monthly budgets with a clean and modern UI.

---

## 📱 Features

- ➕ Add Income & Expense Transactions  
- 📊 View Transaction History  
- 📂 Category-wise Summary  
- 📅 Monthly Reports  
- 💰 Budget Tracking with Progress Bar  
- 🔔 Smart & Funny Notifications (like Swiggy/Zomato 😄)  
- 🎨 Modern UI with Gradient Design  

---

## 🛠️ Tech Stack

- **Language:** Java  
- **IDE:** Android Studio  
- **Database:** SQLite  
- **UI:** XML (Material-style UI)  
  **Version Control:** Git & GitHub
